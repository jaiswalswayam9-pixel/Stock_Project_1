### **Project 3: ML-Powered Stock Predictor**
🤖 **Machine Learning & Predictions**
- **Language**: Python (37.7%), HTML (30.7%), CSS (18.1%), JavaScript (13.5%)
- **Purpose**: Stock price prediction using machine learning models
- **Key Features**:
  - Trained machine learning models (best_model.pkl)
  - Feature scaling and preprocessing
  - Real-time predictions
  - Interactive web interface with JavaScript
- **Key Files**:
  - `app.py` - Flask web app (16.5KB)
  - `train_model.py` - Model training pipeline
  - `predict.py` - Prediction engine
  - `best_model.pkl` - Trained ML model
  - `scaler.pkl` - Feature scaler
- **Technologies**: Scikit-Learn, Flask, JavaScript, D3.js
- **Link**: [View Project](https://github.com/jaiswalswayam9-pixel/Stock_Project_3)

---

### **Project 4: Advanced Time Series Forecasting** ⭐ **Most Advanced**
📈 **Production-Ready Forecasting Engine**
- **Language**: Python (45.9%), HTML (34.3%), CSS (13.8%), JavaScript (6%)
- **Purpose**: Advanced time series forecasting for top 20 stocks
- **Key Features**:
  - Multi-model ensemble forecasting
  - Real-time data fetching for 20+ stocks
  - SQLite database for persistence
  - Interactive dashboard with visualizations
  - Model performance tracking and comparison
  - Risk scoring and recommendations
- **Key Files**:
  - `app.py` - Advanced Flask application (21KB)
  - `train_timeseries_models.py` - Multi-model training (25KB)
  - `forecast_engine.py` - Forecasting logic (13.7KB)
  - `fetch_top20_data.py` - Data pipeline (13.6KB)
  - `quantumforecast.db` - Persistent database (9.8MB)
  - `model_summary.csv` - Model performance metrics
  - `stock_forecasts.csv` - Latest predictions
- **Technologies**: ARIMA, Exponential Smoothing, LSTM, Prophet, Flask, SQLite, JavaScript, D3.js
- **Link**: [View Project](https://github.com/jaiswalswayam9-pixel/Stock_Project_4)
